## 核心问题识别

流式 Markdown 的三个本质矛盾：

1. **不完整性** — token 逐字到来，` ``` python` 可能只收到 ` ``` ` 时就要决定如何渲染
2. **歧义性** — `*` 到来时不知道是 `*bold*`、列表项还是普通字符
3. **回溯性** — 之前的渲染决策可能因后续 token 而需要修正
